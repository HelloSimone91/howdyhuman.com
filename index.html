The Howdy Human Compendium of Values

Discover values in action

This tool helps you explore 100+ human values organized by category with detailed descriptions and real-world examples. Each value is connected to practical "action verbs" that show how these values are demonstrated in daily life.

Use the search bar to find specific values, or explore by categories and tags. Click on a value to learn more about it and discover related values that share similar characteristics.
