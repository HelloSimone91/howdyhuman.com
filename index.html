The Howdy Human Dictionary of Values turns big ideas into everyday actions. Each value comes with verbs, descriptions, and examples to help you notice how it shows up in real life.

The Howdy Human Dictionary of Values

Discover values in action

This dictionary presents values as everyday actions.

Values operate at different levels: personal (how we relate to ourselves), interpersonal (how we relate to others), and social (how we live together). Each entry includes verbs that show how the value is practiced and expressed in real life.

Search for a value or browse by category and verb. Open a card to view a brief description, an example, and related values
